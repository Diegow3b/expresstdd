# expresstdd
Creating Express Project TDD using Supertest with Mocha

## Usage
```shell
> git clone https://github.com/Diegow3b/expresstdd.git
> npm install
> cd expresstdd
```
To run the server at root folder
```shell
./bin/www
```

To runs tests
```shell
npm test
```
